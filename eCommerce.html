Let's do Online Shopping

This project deals with developing an e-commerce website for online shopping. It’s functioning style is quite similar to that of physical retail industry. It provides the user to buy online products such as clothing of their own choice. The main objective of this application is to make it interactive and its ease to use. This project has the functionality to add products to cart form where user can proceed for checkout Technologies used are: HTML5, CSS, Bootstrap, JavaScript, jQuery.
